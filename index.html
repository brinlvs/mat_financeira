Unidade I - Fundamentos

1. Capital e Juro

Capital (C ou PV): valor aplicado ou emprestado.
Juro (J): remuneracao do capital pelo tempo.

2. Relacoes Basicas

M = C + J
i = J / C (taxa unitaria)
n = prazo

3. Regimes de Capitalizacao

  • Simples: juros sobre o capital inicial
  • Compostos: juros sobre juros (exponencial)

4. Fluxo de Caixa

Representacao grafica de entradas (+) e saidas (-) ao longo do tempo.

Unidade III - Descontos Simples

1. Desconto Comercial

D = N * d * n
A = N * (1 - d * n)

2. Relacao d x i

i = d / (1 - d*n)
d = i / (1 + i*n)

3. Prazo Medio

n_medio = Soma(Ni * ni) / Soma(Ni)
Questao 5:
n = (200000*37 + 150000*94 + 62500*52) / 412500 = 60 dias

Unidade IV - Juros Compostos

1. Formula do Montante

M = C * (1 + i)^n
PV = FV / (1 + i)^n

2. Periodos Nao Inteiros

Linear: M = C*(1+i)^p * (1+i*q)
Exponencial: M = C*(1+i)^n

3. Taxas Equivalentes

(1 + i_eq) = (1 + i)^(t_eq/t)

4. Taxa Over (dias uteis)

Fator diario = (1 + i_mensal)^(1/30)
Fator periodo = (Fator diario)^n_du
Questao 4: i = (103,5/100)^(1/2) - 1 = 1,735% a.m.
d = i/(1+i) = 1,705% a.m.

5. Capitalizacao Continua

M = C * e^(r*t)

Unidade V - Taxa Real

1. Taxa Real

(1 + i_real) = (1 + i_nom) / (1 + inflacao)
Aprox: i_real = i_nom - inflacao
